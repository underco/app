# Under App

## The Stack

* React-Native
* Typescript
* Redux
* Immutable
* Jest

## How To Run

See the wiki for environment setup.

Start the device simulator, then start the react-native server:

```
yarn start
```

Then in another terminal build the app:

```
yarn build
```

And run the app in Android:

```
yarn android
```

Or iOS:

```
yarn ios
```
